**src/core/security.py:**
- `from pwdlib import PasswordHash` — use `PasswordHash((Argon2Hasher(),))` (see pwdlib docs)
- `hash_password(password: str) -> str` — returns Argon2id hash
- `verify_password(password: str, hash: str) -> bool` — verifies against hash
- `load_or_generate_rsa_keys(key_dir: Path) -> tuple[bytes, bytes]` — generates 2048-bit RSA key pair if not exists, returns (private_key_bytes, public_key_bytes). Use `cryptography.hazmat.primitives.asymmetric.rsa` for generation, PEM encoding.
- `create_access_token(user_id: str, email: str, private_key: bytes, expires_minutes: int = 15) -> str` — JWT with claims: sub=user_id, email=email, exp=now+expires, iat=now, type="access". Algorithm RS256.
- `verify_access_token(token: str, public_key: bytes) -> dict` — decode with RS256, require ["exp", "sub", "email", "type"]. CRITICAL: pin `algorithms=["RS256"]` to prevent alg:none attack.
- `generate_reset_token() -> str` — `secrets.token_urlsafe(32)`
- `hash_token(token: str) -> str` — `hashlib.sha256(token.encode()).hexdigest()`

**src/services/auth.py:**
- `async def register_user(session: AsyncSession, email: str, password: str) -> User` — check email uniqueness (SELECT first), hash password with `hash_password()`, create User, flush, return. Raise `HTTPException(409)` on duplicate.
- `async def login_user(session: AsyncSession, email: str, password: str, private_key: bytes) -> tuple[str, str]` — find user by email, verify password, create access token via `create_access_token()`, create refresh token (generate raw token, store SHA-256 hash in DB with 7-day expiry), return (access_token, raw_refresh_token). Raise `HTTPException(401)` on failure (generic message: "Invalid email or password" — never reveal which is wrong).
- `async def refresh_tokens(session: AsyncSession, raw_refresh_token: str, private_key: bytes) -> tuple[str, str]` — hash incoming token, find matching non-revoked non-expired RefreshToken in DB, revoke it (set revoked_at=now), create new refresh token (set replaced_by on old), create new access token. Return (new_access, new_refresh). Raise `HTTPException(401)` if token invalid.
- `async def logout_user(session: AsyncSession, raw_refresh_token: str) -> None` — hash token, find and revoke it. No error if not found (idempotent).
- `async def request_password_reset(session: AsyncSession, email: str) -> str | None` — find user by email, if not found return None (don't reveal existence). Generate reset token, hash it, store PasswordResetToken with 1-hour expiry. Return raw token.
- `async def reset_password(session: AsyncSession, raw_token: str, new_password: str) -> None` — hash token, find matching non-expired non-used PasswordResetToken, mark used_at=now, update user password_hash. Raise `HTTPException(400)` if token invalid/expired/used.

**src/api/v1/auth.py:**
Create APIRouter with prefix="/auth", tags=["auth"]. All endpoints:

1. `POST /register` — accepts `RegisterRequest` body. Calls `register_user()`, then `login_user()` to return tokens immediately. Returns 201 + `TokenResponse`. Sets refresh token as httpOnly cookie: `response.set_cookie(key="refresh_token", value=raw_refresh, httponly=True, samesite="strict", secure=True, max_age=7*24*3600, path="/api/v1/auth")`.

2. `POST /login` — accepts `LoginRequest` body. Rate limited with slowapi: `@limiter.limit("5/minute")`. Calls `login_user()`. Returns 200 + `TokenResponse` + refresh cookie (same settings as register).

3. `POST /refresh` — reads `refresh_token` from cookie (`request.cookies.get("refresh_token")`). Calls `refresh_tokens()`. Returns 200 + new `TokenResponse` + new refresh cookie.

4. `POST /logout` — reads refresh_token from cookie. Calls `logout_user()`. Deletes cookie: `response.delete_cookie(key="refresh_token", path="/api/v1/auth")`. Returns 200 + `MessageResponse(message="Logged out")`.

5. `POST /password-reset/request` — accepts `PasswordResetRequest` body. Calls `request_password_reset()`. If token returned, calls `send_password_reset_email()`. ALWAYS returns 200 + `MessageResponse(message="If the email exists, a reset link has been sent")` — never reveal email existence.

6. `POST /password-reset/confirm` — accepts `PasswordResetConfirm` body. Calls `reset_password()`. Returns 200 + `MessageResponse(message="Password reset successful")`.

7. `GET /me` — protected with `Depends(get_current_user)`. Returns 200 + `UserResponse` from current user.

**src/api/deps.py — update `get_current_user`:**
```python
async def get_current_user(
    request: Request,
    session: AsyncSession = Depends(get_db),
) -> User:
    auth_header = request.headers.get("Authorization")
    if not auth_header or not auth_header.startswith("Bearer "):
        raise HTTPException(401, "Not authenticated")
    token = auth_header.split(" ", 1)[1]
    try:
        payload = verify_access_token(token, public_key)  # public_key loaded at startup
    except jwt.ExpiredSignatureError:
        raise HTTPException(401, "Token expired")
    except jwt.InvalidTokenError:
        raise HTTPException(401, "Invalid token")
    user = await session.get(User, uuid.UUID(payload["sub"]))
    if not user or not user.is_active:
        raise HTTPException(401, "User not found or inactive")
    return user
```
